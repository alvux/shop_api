-The API was built using Node.js. Make sure that Node.js and NPM are installed. 
-The information about the database is in the file mlablogin.txt

1. To install the dependencies, enter the command "npm install" at the root of the folder.
2. To start the server, enter the command "npm start".
3. Documentation for the API can be retrieve at http://localhost:8000/.
4. I used Postman to test the API.
